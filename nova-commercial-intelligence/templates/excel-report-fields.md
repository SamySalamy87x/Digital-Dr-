# Campos para Excel Comercial

Este archivo define las columnas recomendadas para generar reportes comerciales descargables en Excel o Google Sheets.

## Hoja 1 — Lead

| Columna | Descripción |
|---|---|
| Fecha | Fecha de generación del reporte |
| Vendedor | Nombre del vendedor o usuario interno |
| Nombre completo | Nombre de quien solicita el análisis |
| Empresa / Cadena | Empresa, cadena hotelera o prospecto |
| Hotel | Nombre del hotel específico |
| Ciudad | Ciudad de operación |
| Estado | Estado de operación |
| Tipo de operación | Boutique, cadena, resort, business class, gran turismo, etc. |
| Categoría | 3, 4, 5 estrellas, gran turismo o pendiente de validar |
| Habitaciones estimadas | Número aproximado de habitaciones |
| Cliente actual | Sí / No |
| Objetivo | Prospección, seguimiento, cotización, ampliación de portafolio, etc. |
| Prioridad | Alta, media o baja |
| Ticket estimado | Monto o rango estimado si existe información suficiente |
| Siguiente acción | Acción comercial recomendada |

## Hoja 2 — Diagnóstico por zona

| Columna | Descripción |
|---|---|
| Zona operativa | Área del catálogo Ecodeli |
| Dolor detectado | Problema operativo identificado |
| Riesgo operativo | Costo, merma, imagen, higiene, experiencia, eficiencia |
| Oportunidad Ecodeli | Cómo se puede ampliar portafolio o resolver el GAP |
| Prioridad | Alta, media o baja |
| Comentarios | Observaciones del vendedor o del agente |

## Hoja 3 — Productos recomendados

| Columna | Descripción |
|---|---|
| Zona | Área operativa |
| Bundle | Nombre del paquete recomendado |
| Clave | SKU o clave de producto |
| Descripción | Descripción del producto |
| Uso | Aplicación operativa |
| Argumento comercial | Ahorro, eficiencia, imagen, higiene o experiencia del huésped |
| Consumo estimado | Consumo sugerido o pendiente de validar |
| Prioridad | Alta, media o baja |

## Hoja 4 — Investigación web

| Columna | Descripción |
|---|---|
| Fuente | Nombre de la fuente |
| URL | Link de consulta |
| Dato encontrado | Hallazgo relevante |
| Tipo de dato | Confirmado, estimado o pendiente de validar |
| Confiabilidad | Alta, media o baja |
| Fecha de consulta | Fecha en que se revisó la fuente |

## Hoja 5 — Seguimiento CRM

| Columna | Descripción |
|---|---|
| Fecha de seguimiento | Fecha programada |
| Responsable | Vendedor o gerente responsable |
| Acción | Llamada, visita, correo, cotización, demo, revisión |
| Estatus | Nuevo, contactado, en diagnóstico, cotizado, negociación, cerrado, perdido |
| Notas | Observaciones del proceso comercial |
